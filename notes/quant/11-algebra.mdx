## Square Identities

1. $(a+b)^2=a^2+2ab+b^2$
2. $(a-b)^2=a^2-2ab+b^2$
3. $(a+b)^2+(a-b)^2=2(a^2+b^2)$
4. $(a+b)^2-(a-b)^2=4ab$
5. $a^2+b^2=(a+b)^2-2ab=(a-b)^2+2ab$
6. $a^2-b^2=(a+b)(a-b)$
7. $a^4+b^4=(a^2+b^2)^2-2a^2b^2$
8. $a^8+b^8=(a^4+b^4)^2-2a^4b^4$

---

## Cube Identities

1. $(a+b)^3=a^3+b^3+3ab(a+b)$
2. $(a-b)^3=a^3-b^3-3ab(a-b)$
3. $a^3+b^3=(a+b)^3-3ab(a+b)=(a+b)(a^2-ab+b^2)$
4. $a^3-b^3=(a-b)^3+3ab(a-b)=(a-b)(a^2+ab+b^2)$

---

## A Special Identity

:::tip Formula
$a^3+b^3+c^3-3abc=(a+b+c)(a^2+b^2+c^2-ab-bc-ca)$
:::

If $a+b+c=0$, then $a^3+b^3+c^3-3abc=0\implies a^3+b^3+c^3=3abc$

This shows that $a,b,c$ are factors of $a^3+b^3+c^3$

---

Again, $a^3+b^3+c^3-3abc$

$=(a+b+c)(a^2+b^2+c^2-ab-bc-ca)$

$=(a+b+c)\dfrac{1}{2}(2a^2+2b^2+2c^2-2ab-2bc-2ca)$

$=\dfrac{1}{2}(a+b+c)[(a-b)^2+(b-c)^2+(c-a)^2]$

:::info Deduction
If $a=b=c$, then $a^3+b^3+c^3-3abc=0$
:::

---

We know that, $(a+b+c)^2=a^2+b^2+c^2+2(ab+bc+ca)$

Also, $a^3+b^3+c^3-3abc=(a+b+c)(a^2+b^2+c^2-ab-bc-ca)$

Combination of these two identities is used in question paper.

## Value Putting Questions

1. Put any value. But the denominator should not be zero.
2. Put value into one variable first.

:::note
If $a^2+b^2+c^2+\dots=0$, then $a=b=c=\dots=0$
:::

## Componendo-Dividendo

If $\dfrac{a}{b}=\dfrac{m}{n}$, then $\dfrac{a+b}{a-b}=\dfrac{m+n}{m-n}$ and vice-versa.

---

## Symmetrical Expression

1. Each variable is treated equally.
2. Same pattern is followed.
3. The positions of any two variables can be interchanged.

### Steps

1. Equally divide the value at the right hand side of the given equation.
2. Put equal value to each variable (e.g. $x=y=z$).

### Example

:::tip Question
If $\dfrac{a}{1-a}+\dfrac{b}{1-b}+\dfrac{c}{1-c}=1$, then $\dfrac{1}{1-a}+\dfrac{1}{1-b}+\dfrac{1}{1-c}=?$
:::

For symmetrical expression, $\dfrac{a}{1-a}=\dfrac{b}{1-b}=\dfrac{c}{1-c}=\dfrac{1}{3}$

Now, $\dfrac{a}{1-a}=\dfrac{1}{3}\implies a =\dfrac{1}{4}$

Therefore, $\dfrac{1}{1-a}=\cfrac{1}{1-\cfrac{1}{4}}=\dfrac{4}{3}$

Hence, $\dfrac{1}{1-a}+\dfrac{1}{1-b}+\dfrac{1}{1-c}=3\times \dfrac{4}{3}=4$

## Minimum Value of Sum

:::tip Question
What is the minimum value of $x+y$ if $xy=16?$
:::

Given that, $xy=16$

| $x$    | $y$     | $x+y$       |  
| :---:  | :----:  | :----:      |
| $1$    | $16$    | $1+16=17$   |
| $2$    | $8$     | $2+8=10$    |
| $4$    | $4$     | $4+4=8$     |

:::info Note
The minimum value of $x+y$ is obtained when $x=y$
:::

---

## Maximum Value of Product

:::tip Question
What is the maximum value of $xy$ if $x+y=20?$
:::

Given that, $x+y=20$

| $x$        | $y$         | $xy$               |  
| :---:      | :----:      | :----:             |
| $1$        | $19$        | $1\times 19=19$    |
| $2$        | $18$        | $2\times 18=36$    |
| $\dots$    | $\dots$     | $\dots$            |
| $10$       | $10$        | $10\times 10=100$  |


:::info Note
The maximum value of $xy$ is obtained when $x=y$
:::

## Higher Orders of a Formula

Suppose, $x+\dfrac{1}{x}=3$

$\implies\left(x+\dfrac{1}{x}\right)^2=3^2$

$\implies x^2+\dfrac{1}{x^2}=9-2=7$

$\implies\left(x^2+\dfrac{1}{x^2}\right)^2=7^2=49$

$\implies\boxed{x^4+\dfrac{1}{x^4}=49-2=47}$

Similarly, $\left(x^3+\dfrac{1}{x^3}\right)$, $\left(x^5+\dfrac{1}{x^5}\right)$, $\dots$ can 
be calculated.

## Some Special Results

:::note Result - 1
If $x>0$, then $x+\dfrac{1}{x}\geq 2$

$\blacksquare$ $x+\dfrac{1}{x}=2\iff x=1$
:::

:::info Result - 2
If $x<0$, then $x+\dfrac{1}{x}\leq -2$

$\blacksquare$ $x+\dfrac{1}{x}=-2\iff x=-1$
:::

:::tip Result - 3
$x^4+x^2y^2+y^4=(x^2+xy+y^2)(x^2-xy+y^2)$
:::

## Another Special Identity

Suppose, $\boxed{x+\dfrac{1}{x}=\sqrt{3}}$

$\implies\left(x+\dfrac{1}{x}\right)^3=(\sqrt{3})^3$

$\implies x^3+\dfrac{1}{x^3}+3\cdot x\cdot\dfrac{1}{x}\left(x+\dfrac{1}{x}\right)=3\sqrt{3}$

$\implies x^3+\dfrac{1}{x^3}+3\sqrt{3}=3\sqrt{3}$

$\implies\boxed{x^3+\dfrac{1}{x^3}=0}$

$\implies x^3=-\dfrac{1}{x^3}$

$\implies\boxed{x^6=-1}$
