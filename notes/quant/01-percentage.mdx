## Percentage More Or Less

:::note Definition
$\textmd{Multiplying Factor} = \dfrac{\textmd{Final}}{\textmd{Initial}}$
:::

| Percentage Change     | Multiplying Factor               |
| :---:                 | :----:                           |
| $20\%\uparrow$        | $1.2=\dfrac{6}{5}$               |
| $30\%\uparrow$        | $1.3=\dfrac{13}{10}$             |
| $25\%\uparrow$        | $1.25=\dfrac{5}{4}$              |
| $100\%\uparrow$       | $2$                              |
| $16.66\%\uparrow$     | $1+\dfrac{1}{6}=\dfrac{7}{6}$    |
| $20\%\downarrow$      | $0.8=\dfrac{4}{5}$               |
| $30\%\downarrow$      | $0.7=\dfrac{7}{10}$              |
| $25\%\downarrow$      | $0.75=\dfrac{3}{4}$              |
| $100\%\downarrow$     | $0$                              |
| $16.66\%\downarrow$   | $1-\dfrac{1}{6}=\dfrac{5}{6}$    |

## Combining Multiple Percentage Changes

:::tip Multiple Percentage Changes
$\large{\blacksquare}\xrightarrow{20\%\uparrow}\Large{\blacksquare}\xrightarrow{30\%\uparrow}\Huge{\blacksquare}$
:::

There are basically three methods:

### Method 1

Assume any value. Let first value $= 100$

The net change in value is : $100\xrightarrow{20\%\uparrow}120\xrightarrow{30\%\uparrow}120+36 = 156$

Therefore the net percentage change : $(156-100)\%=56\%$

---

### Method 2

Calculate `how many times?`

:::note
$20\%\uparrow \implies \dfrac{6}{5}$

$30\%\uparrow \implies \dfrac{13}{10}$
:::

Now, the multiplying factor : $\dfrac{f}{i} = \dfrac{6}{5}\times\dfrac{13}{10} = \dfrac{78}{50} = \dfrac{156}{100}$

Therefore the net percentage change : $\dfrac{f-i}{i}=\dfrac{156-100}{100}\times 100\%=56\%$

---

### Method 3

:::tip Formula
Net Percentage Change $=\left(\pm a\pm b + \dfrac{(\pm a)(\pm b)}{100}\right)\%$

**Use $\mathbf{+}$ sign for increment and $\mathbf{-}$ sign for decrement.**
:::

Here the net percentage change : $\left(20+30+\dfrac{20\times 30}{100}\right)\%=(50+6)\%=56\%$

## To Keep The Product Constant

Suppose $A=B\times C$ and $B\implies 20\%\uparrow=\dfrac{6}{5}$

To keep $A$ constant, we have to multiply $C$ by $\dfrac{5}{6}$

Therefore, $A=\left(B\times \dfrac{6}{5}\right)\times\left(C\times \dfrac{5}{6}\right)$

Thus $C\implies\dfrac{5}{6}\implies \left(1-\dfrac{5}{6}\right)\downarrow=\dfrac{1}{6}\downarrow\implies 16.66\%\downarrow$

---

## Sum of Numbers

Suppose $A=B+C$

Assume $B\implies 10\%\uparrow$ and $C\implies 20\%\uparrow$

To calculate net percentage change on $A$, the ratio $B\ratio C$ must be given. Suppose $B\ratio C=3\ratio 2$

Then net percentage change on $A = \left(\dfrac{3\times 10 + 2\times 20}{3+2}\right)\% = \left(\dfrac{70}{5}\right)\% = 14\%$

---

### Special Case 1

When $B\ratio C=1\ratio 1$

Then net percentage change on $A = \left(\dfrac{1\times 10 + 1\times 20}{1+1}\right)\% = \left(\dfrac{10+20}{2}\right)\% = 15\%$

In this case, simply calculate average percentage change on $B$ and $C$.

---

### Special Case 2

When $B$ and $C$ has same percentage change, we don't need the value of $B\ratio C$ explicitly.
The net percentage change on $A$ is the same as $B$ (or $C$).

Suppose $B\implies 10\%\uparrow$ and $C\implies 10\%\uparrow$.

Then $A\implies 10\%\uparrow$

:::note
$A(10\%\uparrow) = B(10\%\uparrow)+C(10\%\uparrow)$
:::
