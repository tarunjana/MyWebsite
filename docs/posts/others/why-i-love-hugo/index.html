Why I Love Hugo?

January 25, 2021 2-minute read
Tarun Jana
English
Others

I needed a blog for myself. My requirements were pretty straight-forward: A static site generator with flexibility and speed.

I had a little bit of experience with Jekyll. Since I was not familiar with the Ruby programming language, it was quite overwhelming for me. I could go with Gatsby, Next, Gridsome, 11ty, and all these SSG written in JavaScript. But again, I was not a JavaScript developer to deal with these.

Hugo gave me a sigh of relief in that situation. Although I had no experience with Golang, installing Hugo on my system was very easy. I just had to add the path of the hugo.exe in the path variable of my system. And I was ready to create my first Hugo site.

Hugo comes with almost all the necessary features that I need to write technical content.

Here are some features that I use most:
